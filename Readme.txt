How to use the server:
1. Open sbt.bat
2. Type "run" and enter the command line

Before you use it,
You need to set up a database in \conf\application.conf;
You need to specify the location of your training models in \app\edu\isu\adl\classifer\WOSClassifer
Training models are in \data

You can load the project into Eclipse:
1. Open sbt.bat
2. Type "eclipse" and enter the command line
3. Load the project into your Eclipse
